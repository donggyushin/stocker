**VERDICT: ADR-0006 슬리피지 가정 0.1% 유지. costs.py 수정 불필요.**

근거:
1. 실측 중앙값 0.1305% 는 가정 0.1% 보다 30% 높지만, 사전 정의 괴리 기준 (≥0.2% 또는 ≤0.05%) 미달.
2. 슬리피지는 spread 의 절반 (mid→ask 교차) 이므로 실제 단방향 슬리피지 ≈ 0.065%. 백테스트 가정 0.1% 는 보수적 편향 — 실전에 유리한 방향.
3. 유동성·시간대·일간 변동 모두 안정. 구조적 변화 없음.
