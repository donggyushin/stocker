## 관찰

- **stop 2.5% / take 5.0% 일관 우위**: D1·D2 모두 두 서브셋 상위권 공통. 손절 폭 확대 → 세션 내 청산 빈도 ↓ → 평균 보유 시간 ↑ → MDD 얕아짐. ORB 진입 자체의 승률 30%대 구조 불변.
- **force_close_at 15:20 우위**: 마감 5분 전(동시호가 시작 직전) 까지 보유 시 추가 1.5~2%p MDD 개선. 다만 승률·손익비 변화 미미 — 주가가 14:50~15:20 구간에 회복하는 패턴 일부 캡처.
- **승률 × 손익비** 96 조합 모두 0.32 ~ 0.45 범위. 게이트 1.0 까지 격차 2 배 이상.
- KIS API rate limit 워닝 다발하나 진행 영향 없음 (캐시 hit 위주).

## 결론

force_close_at 변경은 ORB 전략의 근본 손실 패턴을 못 깬다. D1 + D2 종합 → **stop 2.5% / take 5.0% 가 전부**, 다른 구조 파라미터 변경은 1~3%p 미세 조정.

→ ORB 본질 가설 ("OR 돌파 = 추세 시작") 이 한국 KOSPI 200 2025-2026 구간에 안 맞는다는 신호 강해짐.
